## キャラクター追加ルール

```
characters/
└── {名前ローマ字}/
    ├── {NAME}.png          ← 最新版（HTMLで使用）
    ├── {NAME}-v1.png       ← 過去バージョン保管
    └── {NAME}-v2.png       ← 過去バージョン保管
```

- ファイル名は **大文字・ハイフン区切り**（例: `YAMADA-SAN.png`）
- 新バージョンを作ったら旧バージョンを `-v番号` でアーカイブ
- HTMLで使う本番ファイルは常に `member-{名前}.png` としてルートにも配置
